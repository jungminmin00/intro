프로젝트 설명

main
React-Router를 사용하여 메뉴 클릭 시 해당 spa로 이동하게 구현하였습니다.
data를 JSON으로 받아와 sort함수를 이용하여 각각을 내림차순으로 만든 후 sass로 스타일을 조정해 상위 4개만 보일 수 있도록 구현하였습니다.
제품 상세, wishList
useState에 있는 제품 데이터를 반복문을 사용하여 구현하였습니다. 검색, 정렬 기능을 구현하기 위해 filter함수와 sort함수를 사용하였습니다. 제품을 클릭하면 filter함수와 Link를 사용해 해당데이터의 slug를 가져오는 기능을 구현했습니다.
useParams()를 사용해 해당 제품의 세부 사항, 리뷰가 출력 되도록 구현하였습니다
‘장바구니 추가’ 버튼 클릭 시 navigate함수를 사용해 wishList페이지로 이동하게 구현했고, localStorage를 사용해 wishList로 보낼 데이터를 저장 후 새로운 배열을 만들어 출력했습니다.
‘지우기’ 버튼 클릭시 제품 slug를 사용해 데이터를 찾은 후 delete연산자로 해당 데이터를 지웠습니다.
